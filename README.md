# 🎬 YouTube Clone

A responsive YouTube-inspired web application built using HTML, CSS, and JavaScript.  
This project replicates the basic user interface and core functionalities of YouTube, including video thumbnails, navigation layout, and embedded video playback.

---

## 🚀 Features

- Responsive homepage layout
- Video thumbnail cards with title and channel details
- Embedded video player using iframe
- Navigation bar design similar to YouTube
- Clean and modern UI
- Dynamic content handling using JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS)
- YouTube Embedded Player (iframe)

## 💡 How It Works

1. The homepage displays multiple video cards.
2. Each card contains a thumbnail image, video title, and channel information.
3. When a user clicks on a video, it opens a video player page.
4. The video is embedded using the iframe tag.
5. JavaScript handles user interactions and page updates.
